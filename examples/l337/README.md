L337 Example
============

This example exists primarily to test the v3 runtime implementation of following documentation:

* [Lando 3 l337 service](https://docs.lando.dev/core/v3/services/l337.html)

Start up tests
--------------

```bash
# should start successfully
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa
lando poweroff
lando start
```
